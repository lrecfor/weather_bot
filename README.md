# weather_bot
weather bot to get the information about the weather

Бот для получения прогноза погоды на сегодня. На запрос /weather и ввод названия города выдает подробный прогноз.

Пример вывода для города Ростов:  
   Город: rostov  
   Время: 09:40  
   Температура: +15  
   Ветер: 1,0 м/с, С  
   Влажность: 93%
